# School Database Class SQL Exercises

## Overview
This repository contains SQL exercise files for a school database class, focusing on MariaDB-compatible syntax. Each file is designed to help students practice database creation, modification, querying, and constraint management.

## Structure
- All files are located in the root directory.
- File names indicate the exercise topic, such as table creation, joins, operations, and constraints.
- No subdirectories or complex project structure.

## Key Files
- `TP SQL sur la jointure 1.sql`, `TP SQL sur la jointure 2.sql`: Practice SQL joins.
- `TP SQL sur les opérations 1.sql`, `TP SQL sur les opérations 2.sql`, `TP SQL sur les opérations 3.sql`: Practice SQL operations.
- `TP N°1  Création d’une base de données et contraintes d’intégrité sous MariaDB.sql`: Database creation and integrity constraints.
- `TP sur la création et modifiction d'une table.sql`: Table creation and modification.
- `Exercice 1 de création de requêtes sans jointure (1).sql`, `TP 1 de création de requêtes sans jointure.sql`, etc.: Query exercises without joins.

## Usage
1. Open any `.sql` file in your preferred SQL client (MariaDB recommended).
2. Review comments for guidance and explanations.
3. Run queries or schema definitions to practice and learn.

## Requirements
- MariaDB or compatible SQL environment.
- Basic knowledge of SQL syntax and database concepts.

## Tips
- Pay attention to comments in each file for hints and explanations.
- Watch for MariaDB-specific syntax differences if using another SQL dialect.
